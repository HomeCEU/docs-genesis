# Course Lookup

## Actors
1. cems admins

## Flow
1. Login
1. Open "Course Lookup"
1. Find course by id or name
1. See Course Detail
    - id
    - name
    - created_on
    - hours
    - Format
    - Disciplines
    - Authors
    - Categories
    - Clinical Topics
    - Owner
    - isActive
    - Instructional Level
    - BOC Instructional Level
    - Total Enrollments
    - Sco filename
